# Hastane Bilgi Yönetim Sistemi (HMS)

Bu proje, Java programlama dili kullanılarak geliştirilmiş bir yönetim sistemidir. Temel amacı, bir hastanedeki doktor ve hasta kayıtlarını dijital ortamda yönetmek, sorgulamak ve raporlamaktır.

## Özellikler
- **Hasta Kayıt Yönetimi:** Hasta bilgilerinin sisteme eklenmesi ve takibi.
- **Doktor Paneli:** Doktorların uzmanlık alanlarına göre listelenmesi.
- **Hata Yönetimi (Exception Handling):** `KisiBulunamadiException` gibi özel hata sınıfları ile güvenli veri sorgulama.
- **Polimorfizm ve Kalıtım:** `Kisi` ana sınıfından türetilen `Hasta` ve `Doktor` sınıfları ile OOP yapısı.

## Kullanılan Teknolojiler
- **Dil:** Java
- **Mimari:** Nesne Yönelimli Programlama (OOP)
